# TRE2026 Mobile Web Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build a static, mobile-first TRE2026 planning web app that reads local JSON, shows purchased and candidate activities, and analyzes schedule conflicts including Gold Card and Silver Card benefit impacts.

**Architecture:** Use a Vite + React + TypeScript frontend with local JSON data modules, Zod-based validation, a small domain layer for normalization and conflict detection, and three mobile tabs for schedule, conflict analysis, and catalog browsing. Keep the app static so build output can be copied directly to a phone.

**Tech Stack:** Vite, React, TypeScript, Tailwind CSS, Vitest, Testing Library, Zod
